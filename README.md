# ScaleBay Blog Static

Независимая статическая версия блога ScaleBay без Astro и сборки.

## Файлы

- `index.html` - независимая статическая страница с двумя статьями в ленте.
- `styles.css` - оформление.
- `assets/` - локальные изображения статей.

## Запуск

Откройте `index.html` в браузере или опубликуйте папку как обычный static site.
